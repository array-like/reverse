# Installation

Can be managed using
[yarn](https://yarnpkg.com/en/docs),
[npm](https://docs.npmjs.com),
or [jspm](https://jspm.org/docs).


### yarn
```terminal
yarn add @array-like/reverse
```

### npm
```terminal
npm install @array-like/reverse --save
```

### jspm
```terminal
jspm install npm:@array-like/reverse
```
